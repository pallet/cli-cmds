com.palletops.cli.resolve documentation

Functions for resolving and finding cli commands

command-map

var

A sequence of command namespace symbols.

commands

fn

[context]

Return a sequence of the available command namespace symbols. This does not load the namespaces.

filter-command-prefix

fn

[cmds cmd-path]

Filter a map of commands for commands with the given prefix.

find-command

fn

[context cmd-path cmd & {:keys [not-found-f], :or {not-found-f command-not-found}}]

Return the command function for the given context and cmd name.

ns-plugins

fn

[ns-sym]

Return a sequence of [command, namespace symbol] tuples.

prefix-plugins

fn

[prefix]

Return a sequence of [command, namespace symbol] tuples.

resolve-command

fn

[context cmd-path cmd]
[context cmd]

Return the command function for the given context and cmd name. Returns nil if the command can not be resolved. cmd-path is a dotted namespace path for sub-commands.

resolve-command-from-ns

fn

[ns-sym cmd-path cmd]

Require the given namespace symbol, and then resolve the cmd symbol.

resolve-main

fn

[context]

Resolve a main entry point.

suggestions

fn

[cmd cmds]

Suggest possible misspellings for command from list of commands.
